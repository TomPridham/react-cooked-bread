# Quick Start

### Provider

Add the toast provider to a top-level position in your app.

#### app.tsx

```tsx
import {
  ToastProvider, // Provider
  DefaultToastRoot, // Toast shell (animates transitions)
  DefaultToastContent, // Inner content (icons, buttons, text, color)
} from 'react-cooked-bread'

const MyApp = () => (
  <ToastProvider toastRoot={DefaultToastRoot} toastContent={DefaultToastContent}>
    ...
  </ToastProvider>
)
```

### Toast

Now, create a toast!

#### {component}.tsx

```tsx
import { useToasts } from 'react-cooked-bread'

const Form = () => {
  const { addToast } = useToasts()

  const handleSubmit = async () => {
    // ...
    addToast('Form submitted successfully!', {
      type: 'success',
    })
  }

  return <form onSubmit={handleSubmit}>{/* ... */}</form>
}
```
